# 🌐 Projeto WeCode 

![Status](https://img.shields.io/badge/Status-Concluído-success)

## 💻 Sobre o Projeto
Projeto de uma página web desenvolvida para aplicar conceitos de níveis básicos e intermediários de estilização e estruturação.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3

## 🚀 Como Executar
1. Clone este repositório.
2. Abra a pasta do projeto no seu editor de código.
3. Inicie a aplicação executando o arquivo principal com a extensão **Live Server**.
